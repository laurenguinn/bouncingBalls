# Random Bouncing Balls

**Name:** Random Bouncing Balls

**Description:** This simple program creates 5 balls of random color and that bounce across the screen at randomly generated speeds.  The color of the balls and the background of the screen change at a set interval of time as long as the program is running. 

**Usage:** Educational/entertainment purposes.
